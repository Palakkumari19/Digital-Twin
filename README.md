## 🧠 Digital Memory Twin
_Learn by Active Recall, Not Passive Reading_

Digital Memory Twin is an AI-powered knowledge system that transforms raw notes
into a structured, interactive memory network. Instead of manually searching
through large documents, users can ask questions, explore concept relationships
visually, and learn using active recall.

## Problem Statement
Students and professionals work with large amounts of unstructured notes, but
existing tools rely on manual search or passive reading. This leads to poor
context retention, inefficient revision, and limited learning outcomes.
Key challenges:
- Time-consuming information retrieval
- Loss of context during revision
- Passive learning with low retention
  
## Solution
Digital Memory Twin converts unstructured notes into an explainable knowledge
system that supports:
- Deterministic question answering
- Visual exploration of concept relationships
- A foundati on for active-recall-based learning
The system prioritizes correctness, transparency, and extensibility over black-
box behavior.

## 💡 Why It’s Different
Most AI tools help you access information.
Digital Memory Twin helps you internalize and remember it.

## We combine:
- Retrieval
- Visualization
- Memory anchors
- Self-testing
- Into one unified cognitive system.

## Core Features

**Implemented**
- Upload and process raw text notes
- Sentence-level chunking
- Entity extraction for concept mapping
- Keyword-based question answering
- Real-time knowledge graph visualization
- Clean frontend–backend separation
  
**Learning-Oriented Design**
- Answers are derived strictly from user-uploaded content
- No external data sources
- Designed to extend into flashcards and recall-based learning

## 🏗 Tech Stack

**Frontend**
React + TypeScript, Tailwind CSS, react-force-graph-2d

**Backend**
FastAPI, Azure OpenAI, FAISS (Vector Search), MongoDB

## 🎯 Target Users

- Students preparing for exams
- Researchers reviewing papers
- Legal & compliance professionals
- Knowledge-intensive teams

## Why This Matters

Digital Memory Twin shifts knowledge systems from passive consumption to
structured understanding and recall. By combining retrieval, visualization, and
learning workflows, it transforms notes into an interactive memory companion.
