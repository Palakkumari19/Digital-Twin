**🧠 Digital Memory Twin**
_Turn Documents Into Structured Memory_

Digital Memory Twin transforms static PDFs into an interactive learning system.
Instead of just answering questions, it helps users understand, visualize, and retain knowledge.

**🚀 What It Does**

🔎 Semantic Q&A using Retrieval-Augmented Generation (RAG)
🕸 Interactive Knowledge Graph visualization
📇 Auto-generated Flashcards for quick revision
🧪 Conceptual Quiz generation with active recall
🧠 Structured memory reinforcement loo

**💡 Why It’s Different**

Most AI tools help you access information.
Digital Memory Twin helps you internalize and remember it.

We combine:
Retrieval
Visualization
Memory anchors
Self-testing
Into one unified cognitive system.
